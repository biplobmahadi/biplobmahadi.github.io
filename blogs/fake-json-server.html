Fake Json Server
Published: Aug 13, 2022 | Last Modified: Aug 13, 2022
As a frontend engineer we always need data as REST or GraphQL or gRPC API from a backend server. But it's time consuming to create a backend service to practice our frontend things.
So, we can create a fake REST API server with zero coding in less than 30 sec using JSON Server to practice frontend things.
We can make GET, POST, PUT, PATCH and DELETE requests, changes will be automatically and safely saved to db.json using lowdb.
JSON Server can support Filter, Sort, Search, Pagination and other necessary things.
Let's start ...

Start JSON Server: json-server --watch db.json --port 4000
JSON Server provides all request:
GET /posts
GET /posts/1
POST /posts
PUT /posts/1
PATCH /posts/1
DELETE /posts/1
Request with Filter, Sort, Search, Pagination:
Filter /posts?title=json-server&author=typicode
Sort /posts?_sort=author&_order=asc or desc
Pagination /posts?_page=1&_limit=10
Search /posts?q=axios
We can achieve everything using JSON Placeholder. No need db.json.
Load remote schema: json-server http://jsonplaceholder.typicode.com/db --port 4000
Here we have /db, using /db data will persist for a single session.
We can also create our own middleware to add some extra power.
But the best part is: Module
We can use the json-server project as a module in combination with other Express middlewares.
Here, we will create a project using json-server only to get responses in specific format.
Every response will be like { isSuccess: boolean; data: actual data; message: string; }
Create a node project using npm init -y
Install JSON Server: npm install json-server
Create server.js
1

2
const jsonServer = require("json-server");
3


4
const server = jsonServer.create();
5
const router = jsonServer.router(path.join(__dirname, "db.json"));
6
const middlewares = jsonServer.defaults();
7


8
server.use(middlewares);
9
server.use(router);
10


11
router.render = (_req, res) => {
12
  const { statusCode, locals: { data } } = res;
13


14
  res.jsonp({
15
    isSuccess: (statusCode >= 200 && statusCode < 300) ? true : false,
16
    data: (statusCode >= 200 && statusCode < 300) ? data : null,
17
    message: (statusCode >= 200 && statusCode < 300) ? "Successfully done!" : "Error occurred!",
18
  });
19
};
20


21
const port = 5050;
22


23
server.listen(port, () => {
24
