Axios Instances
Published: Jul 28, 2022 | Last Modified: Jul 31, 2022
We can create a new instance of axios with a custom config.
We will create 2 axios instances for our project. One for public requests which does not need any authorization and another for private requests which need an authorization header.
We can use only one instance, when we add authorization header using axios interceptor then every request will add the authorization header. So we need 2 different instances to request with the authorization header and without.
Public Instance
Create instances/publicInstance.ts
1
import axios, { AxiosInstance } from "axios";
2


3
export const axiosPublicInstance: AxiosInstance = axios.create({
4
  baseURL: 'Your API Base URL',
5
});
Private Instance
Create instances/privateInstance.ts
1
import axios, { AxiosInstance } from "axios";
2


3
export const axiosPrivateInstance: AxiosInstance = axios.create({
4
  baseURL: 'Your API Base URL',
5
});
Create a instances/index.ts file to export everything from instances folder

2
